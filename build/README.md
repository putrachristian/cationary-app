# PWA Icons

Folder ini berisi ikon-ikon yang diperlukan untuk PWA (Progressive Web App).

## Ikon yang Diperlukan

Untuk PWA berfungsi dengan baik, Anda perlu menambahkan ikon berikut:

1. **pwa-64x64.png** - Ikon 64x64 pixels
2. **pwa-192x192.png** - Ikon 192x192 pixels
3. **pwa-512x512.png** - Ikon 512x512 pixels
4. **maskable-icon-512x512.png** - Ikon maskable 512x512 pixels
5. **apple-touch-icon.png** - Ikon untuk iOS (180x180 pixels)
6. **favicon.png** - Favicon standar (32x32 atau 64x64 pixels)
7. **favicon.ico** - Favicon untuk browser lama

## Cara Membuat Ikon

1. Siapkan logo aplikasi dengan resolusi tinggi (minimal 512x512)
2. Gunakan tool online seperti:
   - https://realfavicongenerator.net/
   - https://www.pwabuilder.com/imageGenerator
   - https://favicon.io/
3. Atau gunakan ImageMagick/Photoshop untuk membuat berbagai ukuran

## Catatan

Ikon-ikon ini akan digunakan oleh browser dan sistem operasi untuk menampilkan aplikasi saat diinstall sebagai PWA.
